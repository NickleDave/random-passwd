(Really) Random Passwd
======================

This simple Python class uses the random.org api
to generate really secure passwords.

Example Use
===========

# Assign the class to p
p = RandomPasswd()

# Generate and print a password of length 8
# with a level 2 security.
print p.passwd(8, 2)
